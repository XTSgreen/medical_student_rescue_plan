<---如你所见，这是目前互联网前端开发最常见且最原始的文件形式，html文件，这行代码左右两端的大于号、小于号和横杠是html文件的注释形式---> <---简单的说，前端开发就是负责创建网站或网页类应用中用户所能看到的那一部分的工程---> <---例如你在bilibili.com中能看到的布局、按钮、颜色、图片等可视化界面，以及你对网站进行的点击、滑动、输入等交互过程，这些都是前端开发涵盖的部分---> """ 前端开发依赖的核心技术： """ <---前端开发主要依赖三种基础技术:---> <---1,HTML(HyperText Markup Language) 规范属性：一种基于SGML的标记语言，当前由WHATWG维护的HTML Living Standard定义，W3C同步发布快照版本。其核心是定义文档的结构化语义与元素类型，通过标签构建内容层次。 解析模型：浏览器解析 HTML 生成 DOM（文档对象模型）树，作为后续样式计算、布局和脚本执行的底层数据接口。HTML 元素支持全局属性和内容模型，确保语义化与可访问性。 工程化角色：作为 Web 应用的骨架，与 CSSOM 结合形成渲染树，同时承载前端框架的挂载点。现代 HTML 还支持 Web Components（自定义元素、Shadow DOM）以实现组件化封装。---> <---2,CSS（层叠样式表）：负责网页的样式。就像给房子装修，用来设置颜色、字体、大小、布局、动画等。例如让标题变成红色，把导航栏固定在顶部。---> <---3,JavaScript：负责网页的行为。就像给房子装上电灯、开关，让页面“动起来”并能响应用户的操作。例如点击按钮弹出提示框、轮播图自动切换、表单验证输入是否为空。---> <---我相信以各位医学生贫瘠的计算机知识压根就没听懂上面这三行都是什么东西---> <---没关系，上面只是定义，就像你上了临床在按压麦氏点时不会背解剖书，而是会"对着这块按一把"--->
