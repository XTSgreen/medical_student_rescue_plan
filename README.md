# 医学生自救计划

## 项目介绍

在医疗系统逐步腐烂而ai编程能力逐步提升的当下，希望这个项目能帮助一部分医学生/计算机初学者找到出路，该项目中的绝大部分代码将由各种中国大陆能简便访问的ai生成，例如glm、minimax、qwen等，涵盖互联网开发、原生开发、深度学习、机器学习、硬件开发等的基础知识，起到帮助入门的作用

## 使用方法：

将这个文档拖进各种编程软件，在这里推荐trae，注意别下错了，用microsoft edge的必应搜索时记得好好观察一下是否为广告，是广告的话一般会标出来；推荐trae的原因是这东西有ai功能，并且可以免费体验glm、minimax、qwen等模型，这个顺序也是我推荐的使用优先级，但是要注意，minimax作为一个小参数量模型，不适合在大上下文或者复杂实现逻辑里应用

拖进trae并加载完成后，左侧会显示文件结构，可以在此处选择自己希望访问的文件并进行编辑
此外，这个项目也是我自己学习git操作和各种计算机知识的日志，祝我们共同进步
